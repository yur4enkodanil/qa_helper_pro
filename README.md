# QA Helper Pro 🚀 [v2.0 Stable]

**QA Helper Pro** — это комплексный швейцарский нож для инженера по тестированию, разработанный для автоматизации рутинных задач, проектирования тест-кейсов и визуального контроля качества.

Инструмент объединяет возможности генерации данных, анализа целостности и визуального регрессионного тестирования в одном удобном веб-интерфейсе.

---

## 🛠 Стек технологий

Проект построен на современном стеке Python для обеспечения скорости и стабильности:

* **Интерфейс:** [Streamlit](https://streamlit.io/) (динамическое веб-приложение)
* **Анализ данных:** Pandas, NumPy, Itertools
* **Генерация:** Faker
* **Компьютерное зрение:** OpenCV, Pillow (PIL)
* **Системные утилиты:** Tkinter (нативные диалоги), Hashlib

---

## 📋 Основные модули

### 📊 Тест-дизайн и данные
* **Matrix Lab:** Создание комбинаторных матриц. Поддержка методов **Full Combinatorial** и **Pairwise** для оптимизации количества проверок.
* **Data Generator:** Мгновенная генерация имен, email, адресов и других данных через Faker.
* **JSON Lab:** Инструмент для форматирования и валидации JSON-ответов API.

### 👁️ Визуальный регресс
* **Сравнение скриншотов:** Попиксельное сравнение «Эталона» и «Текущего результата» с генерацией тепловой карты (Heatmap) различий.
* **База эталонов:** Встроенная система хранения и управления (загрузка/удаление) эталонными изображениями.

### 📁 Файловые операции
* **Файловый цех:** Генерация деревьев папок любой вложенности и наполнение их файлами заданного веса (от КБ до ГБ) для нагрузочного тестирования.
* **Integrity Lab:** Быстрый расчет контрольных сумм файлов (SHA-256, MD5) для проверки целостности.

## 🚀 Быстрый запуск

### Вариант 1: Для Windows (рекомендуется)
Просто запустите файл `run_app.bat`. Скрипт автоматически:
1. Создаст виртуальное окружение (`venv`).
2. Установит все необходимые зависимости.
3. Запустит локальный сервер приложения.

### Вариант 2: Ручная установка
Воспользуйся инструкцией в УСТАНОВКА_QA_HELPER
