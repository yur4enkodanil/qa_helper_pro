# QA Helper Pro 🚀 [v2.2.1 Stable]

**QA Helper Pro** — это ультимативный комбайн для Middle QA инженера, разработанный для автоматизации рутинных задач, ускорения проверок и повышения качества тестирования.

Инструмент объединяет более 20 утилит для тестирования Web, API, Mobile и анализа данных в одном удобном веб-интерфейсе.

## 🛠 Стек технологий

Проект построен на современном стеке Python для обеспечения скорости и стабильности:

* **Интерфейс:** [Streamlit](https://streamlit.io/) (динамическое веб-приложение)
* **Анализ данных:** Pandas, NumPy
* **Веб-автоматизация:** Playwright
* **Компьютерное зрение:** Pillow (PIL), EasyOCR
* **HTTP-запросы:** Requests

---

## 📋 Основные модули

### 🌐 Инструменты для Веб
* **Эмулятор экранов:** Проверка адаптивности под любые мониторы и мобильные устройства.
* **Frontend Анализатор:** Поиск битых ссылок и проблем с доступностью (Accessibility).
* **UI Инспектор:** Распознавание текста (OCR), симулятор "челок", инспектор шрифтов и линейка.
* **Сравнение скриншотов:** Pixel Perfect анализ визуальных регрессий.

### 📡 Инструменты для API и Данных
* **API Клиент:** Отправка HTTP-запросов (GET, POST и др.) с импортом из cURL.
* **Повторитель запросов:** Базовое нагрузочное тестирование эндпоинтов.
* **Генератор данных:** Создание реалистичных ФИО, ИНН, адресов, карт и т.д.
* **JSON Lab:** Валидация, форматирование и "причесывание" сложных JSON.

### 📁 Инструменты для Файлов
* **Файловый цех:** Генерация файлов любого веса и формата для проверки лимитов загрузки.
* **Анализатор логов:** Мощная фильтрация и анализ больших лог-файлов.
* **Проверить хеш:** Расчет контрольных сумм (MD5, SHA1, SHA256, SHA512) для проверки целостности.
* **Визуальная проверка:** Управление базой эталонных скриншотов и их сравнение с актуальными.

### 📱 Мобильная лаборатория
* **QR-код Хаб:** Генерация и сканирование QR-кодов.
* **Облачный буфер:** Быстрая передача текста с ПК на телефон.
* **Гайд по разрешениям:** Справочник по изменениям в разрешениях Android и iOS.
* **Тестер Deep Link:** Проверка диплинков и универсальных ссылок.

## 🚀 Быстрый запуск

### Для Windows
1.  **Установка:** Запустите `install.bat`. Скрипт автоматически создаст виртуальное окружение (`venv`) и установит все зависимости. Это нужно сделать только один раз.
2.  **Запуск:** Используйте `start.bat` для запуска приложения.

### Ручная установка (Linux/macOS/Windows)
1.  Убедитесь, что у вас установлен Python 3.10+.
2.  Создайте и активируйте виртуальное окружение:
    ```bash
    python -m venv venv
    source venv/bin/activate  # Для Linux/macOS
    .\venv\Scripts\activate   # Для Windows
    ```
3.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    playwright install
    ```
4.  Запустите приложение:
    ```bash
    streamlit run main.py
    ```
